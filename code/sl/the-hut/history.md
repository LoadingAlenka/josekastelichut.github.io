# Zgodovina koče

### Začetki

Gradnja se je začela leta 1950, vodila jo je skupina mladih, ki so želeli smučati v Los Penitentes. Sodelovali so mnogi, a pet glavnih: ..., ..., ..., ..., in vodja in gonilna sila projekta: Peter Bajda.

### Prenos v slovensko skupnost Mendoza

Čeprav so bili mladi člani skupnosti, niso imeli neposrednega nadzora nad kočo vse do njenega prenosa leta 19...

### Današnji čas

Odprta je za javnost in služi kot zakladnica neštetih zgodb, počitnic, pohodov in pesmi ljudi z vsega sveta.

#### 50. obletnica koče

Dogodek, ki so ga praznovali v začetku avgusta, je združil vse, ki so sodelovali pri njeni gradnji in vzdrževanju, spali na njenih odejah ali preprosto šli skozi njena vrata.
